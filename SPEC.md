# Obsidian 知识库网站项目文档

## 1. 项目概述

- **项目名称**: Obsidian Knowledge Hub
- **项目类型**: 静态文档网站
- **核心功能**: 将Obsidian笔记库发布为可搜索、双向链接的静态网站
- **目标用户**: 技术开发者、个人知识管理

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 静态网站生成 | 将Obsidian markdown文件转换为静态HTML | P0 |
| 全文搜索 | 实时搜索笔记标题和内容 | P0 |
| 双向链接展示 | 显示引用此文档的页面（ backlinks） | P0 |
| 双向链接导航 | 支持 [[wiki-link]] 语法跳转 | P0 |
| 文件夹导航 | 侧边栏展示目录结构 | P1 |
| 标签系统 | 支持 #tag 标签分类和筛选 | P1 |
| 图谱可视化 | 展示页面关联关系（可选增强） | P2 |

### 2.2 用户交互

- 搜索框即时搜索
- 点击链接无缝跳转
- 底部展示反向链接
- 支持暗色/亮色主题切换（默认暗色赛博朋克）

---

## 3. 技术方案

### 3.1 技术选型

- **框架**: Vue 3 + Vite 或 React + Next.js
- **静态生成**: 支持 Obsidian 导出的工具，如 obsidian-html 或自定义脚本
- **搜索**: FlexSearch 或 Fuse.js（客户端搜索）
- **样式**: Tailwind CSS

### 3.2 目录结构

```
/
├── docs/                  # Obsidian 导出的 markdown
├── src/
│   ├── components/        # 组件
│   ├── pages/             # 页面
│   ├── utils/             # 工具函数
│   └── styles/            # 样式
├── public/                # 静态资源
└── index.html             # 入口
```

---

## 4. UI/UX 设计

### 4.1 主题风格

- **风格**: 赛博朋克（Cyberpunk）
- **配色方案**:
  - 主背景: `#0d0d0d` (深黑)
  - 次背景: `#1a1a2e` (深蓝黑)
  - 主色调: `#00fff5` (霓虹青)
  - 辅助色: `#ff00ff` (霓虹粉)
  - 文字色: `#e0e0e0` (浅灰)
  - 链接色: `#00fff5` (霓虹青)
  - 强调色: `#ffff00` (霓虹黄)

### 4.2 布局设计

```
┌─────────────────────────────────────────────────┐
│  Header: Logo + 搜索框 + 主题切换                │
├──────────┬──────────────────────┬───────────────┤
│          │                      │               │
│  Sidebar │     Main Content     │  Backlinks    │
│  目录导航 │     笔记内容          │  反向链接     │
│          │                      │               │
└──────────┴──────────────────────┴───────────────┘
```

- **Header**: 固定顶部，包含搜索和主题切换
- **Sidebar**: 左侧目录树，支持展开/折叠
- **Main**: 中央内容区，渲染 markdown
- **Backlinks**: 右侧展示引用当前页面的其他页面

### 4.3 视觉特效

- 霓虹发光效果（text-shadow, box-shadow）
- 渐变边框
- 轻微的扫描线效果（可选）
- 流畅的页面过渡动画

---

## 5. 页面清单

| 页面 | 路由 | 描述 |
|------|------|------|
| 首页 | `/` | 入口页，展示最近笔记和搜索 |
| 笔记页 | `/:slug` | 渲染单篇笔记内容 |
| 标签页 | `/tags` | 标签索引 |
| 标签详情 | `/tags/:tag` | 该标签下的所有笔记 |
| 搜索结果 | `/search` | 搜索结果列表 |

## 🔥 Top 5 深度分析

### 1️⃣ obra/superpowers - Agentic 技能框架

**项目地址**: https://github.com/obra/superpowers

⭐ **100,399** | 📈 **周增 19,176**

### 📖 简介

Agentic 技能框架 & 软件开发方法论。让你的编码 Agent 具备完整的开发工作流，从需求分析到代码审查全流程自动化。

### 🏗️ 核心架构

```
Superpowers 工作流：
│
├── 阶段1: 需求分析 (brainstorming)
│   └── 通过提问细化需求，形成设计文档
│
├── 阶段2: 设计验证
│   └── 分块展示设计，獲取确认
│
├── 阶段3: 实施计划 (writing-plans)
│   └── 分解为 2-5 分钟的小任务
│
├── 阶段4: 自动化开发 (subagent-driven-development)
│   ├── 调度子 Agent 执行每个任务
│   └── 两阶段审查（规范合规 → 代码质量）
│
└── 阶段5: 完成交付 (finishing)
    └── 验证测试、合并/清理
```

### 🎯 核心技能

| 技能 | 功能 |
|------|------|
| **brainstorming** | 代码前需求分析，Socratic 提问 |
| **writing-plans** | 详细实施计划 |
| **subagent-driven-development** | 子 Agent 驱动开发 |
| **test-driven-development** | 红绿重构 TDD |
| **requesting-code-review** | 代码审查 |
| **systematic-debugging** | 系统化调试 |

### 💡 核心理念

- **Test-Driven Development** - 先写测试
- **Systematic over ad-hoc** - 流程 > 猜测
- **Complexity reduction** - 简单即美
- **Evidence over claims** - 验证再宣布成功

### 🔧 支持平台

- ✅ Claude Code (插件市场)
- ✅ Cursor
- ✅ Codex
- ✅ OpenCode
- ✅ Gemini Extensions
